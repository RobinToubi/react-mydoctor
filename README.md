## Installation

Pour installer l'application il vous suffit tout simplement de lancer la commande suivante :
```npm install```

Une fois les modules installés, vous pouvez lancer la solution avec la commande suivante :
```npm start```

## Infos

L'application suit l'architecture suivante :
- public/ : images.. de l'app
- src/ : application
- src/styles : explicite
- src/components : components React classés

## Warning

Afin d'assurer que l'uilisateur est bien authentifié, utiliser la route d'API `/api/user/verify`.
